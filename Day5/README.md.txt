# Day 05 - Encoder & Decoder in Verilog

## Objective
Learned Encoder, Priority Encoder, and Decoder design using Behavioral Modeling in Verilog HDL.

---

## Concepts Covered
- Encoder
- Priority Encoder
- Decoder
- One-hot encoding
- always @(*)
- if-else statement
- case statement

---

## Theory

### Encoder
Converts multiple input lines into binary output.

### Priority Encoder
Selects the highest priority active input.

### Decoder
Converts binary input into one-hot output.

---

## Programs Implemented

### 1. 8:3 Encoder
Encodes 8 input lines into 3-bit binary output.

### 2. 8:3 Priority Encoder
Provides priority to higher-order inputs.

### 3. 3:8 Decoder
Converts 3-bit binary input into one-hot 8-bit output.
