## 🧠 Theory

### Register
A register stores multiple bits of data.

Flip-Flop:
- stores 1 bit

Register:
- stores multiple bits

---

### Shift Register
Moves data left or right
on every clock pulse.

Used in:
- serial communication
- processors
- datapaths
